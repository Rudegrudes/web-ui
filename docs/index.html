Browser-Use Web UI

Uma interface web para um agente de IA que interage com navegadores web para realizar tarefas automatizadas.

Recursos Principais

  • • Controle de navegador web por IA
  • • Suporte a múltiplos modelos de LLM
  • • Visualização em tempo real via VNC
  • • Interface web personalizável
  • • Sessão de navegador persistente

Modelos Suportados

  • • OpenAI
  • • Anthropic
  • • Google
  • • DeepSeek
  • • Ollama (local)
  • • Mistral AI
